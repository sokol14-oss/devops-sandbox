# devops-sandbox
### Infrastructure & Automation Tools
### Репозиторий с моими наработками по автоматизации серверов.
### Что внутри:
* ###  Telegram MTProto Proxy: Развернут в Docker для обхода DPI-блокировок. Используется порт 8443 для сосуществования с Nginx
* ###  Bash Scripts: Скрипты для автоматической установки и проверки состояния сетевых сервисов (Dante, iptables)
 ###  Стек: Linux (Ubuntu), Docker, Docker Compose, Bash, Networking (Troubleshooting IPv4/IPv6).
